# 🧠 Algorithm & CS Study Repository

알고리즘 문제 풀이와 CS 이론 정리를 기록하는 저장소입니다.  
매일 조금씩이라도 **꾸준히 성장하기**를 목표로 합니다.

<br>

## 📚 구성

| Directory | Description |
|----------|-------------|
| `boj/` | 백준 문제 풀이 코드 |
| `programmers/` | 프로그래머스 문제 풀이 코드 |
| `swea/` | SWEA 문제 풀이 코드 |
| `theory/` | 자료구조, 알고리즘, CS 이론 정리 |
| `tips/` | 자주 까먹는 문법/개념/테크닉 정리 |

<br>

## 🧩 문제 풀이 목표

- 문제를 단순히 **맞히는 것**보다  
  `왜 이렇게 풀었는가?`, `다른 접근법은 무엇인가?` 를 중심으로 기록
- 하루 최소 **1문제 이상 풀기** (가능한 범위에서 꾸준하게)
- 틀린 문제 / 시간이 오래 걸린 문제는 `review.md`로 별도 정리

<br>

## 🛠 사용 언어 및 환경

- Language: Python

## 🔥 Commit Convention

| Prefix | Meaning |
|-------|---------|
| `solve:` | 문제 풀이 업로드 |
| `fix:` | 코드 수정 |
| `docs:` | 이론 정리 문서 추가 또는 수정 |
| `refactor:` | 성능 개선 또는 코드 구조 개선 |
| `chore:` | 폴더/환경/설정 등 잡다한 수정 |
